Understanding I/O Statistics
Context Types
normal
Standard buffer pool operations
vacuum
Operations during vacuum processes
bulkread
Large sequential reads (e.g., COPY, bulk scans)
bulkwrite
Large sequential writes
Object Types
relation
Tables and indexes
temp relation
Temporary tables used during query processing
Key Metrics
Reads
Number of read operations that went to storage
Hits
Reads satisfied from the buffer cache
Hit Ratio
Percentage of reads from cache (higher is better)
Writes
Number of write operations to storage
Evictions
Buffers evicted to make room for new data
Fsyncs
Synchronous writes to durable storage
