Understanding Data Lineage
Event Triggers

Event triggers fire on DDL events (CREATE, ALTER, DROP). They can be used to audit schema changes or enforce policies.

Foreign Keys

Foreign key relationships show data dependencies between tables. CASCADE rules indicate automatic propagation of changes.

Impact Analysis

Use dependencies to understand the impact of schema changes. Views and functions may break if referenced tables are modified.
