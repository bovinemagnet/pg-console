About

Generate comprehensive schema documentation for your PostgreSQL database. The documentation includes:

  • Tables with columns, types, and defaults
  • Primary keys, foreign keys, and constraints
  • Indexes with columns and types
  • Views with SQL definitions
  • Functions with signatures and properties
  • Custom types (ENUMs, composites, domains)
  • Sequences with parameters
  • Comments from pg_description
Tip: Use AsciiDoc format for integration with Antora documentation sites.
Output Formats
HTML

Styled, self-contained HTML with table of contents. Perfect for printing or sharing as a standalone file.

Markdown

GitHub-flavoured Markdown for version control, README files, and wiki pages.

AsciiDoc

AsciiDoc format with TOC and syntax highlighting. Ideal for Antora documentation sites.
